# ds-ml-projects
This repository contains data‑science, machine‑learning, and deep‑learning notebooks built as part of **Scaler's Data Science and Machine Learning program** and specialization/courses from **DeepLearning.ai**.

## Scaler's DSML program

Each case study demonstrates practical application of core techniques—EDA, statistical inference, feature engineering, regression, classification, clustering, time‑series forecasting, computer vision, NLP, and recommender systems—on real business problems.  The notebooks are written in Python using libraries such as pandas, NumPy, scikit-learn, statsmodels, TensorFlow/Keras, seaborn, matplotlib, plotly and Hugging Face Transformers.
